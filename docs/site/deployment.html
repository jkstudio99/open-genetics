# Deployment

Apache .htaccess

RewriteEngine On
RewriteBase /open-genetics/public/

RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php [QSA,L]

Production Checklist

# Item Action
1 APP_DEBUG=false ซ่อน error details จาก client
2 Strong JWT_SECRET openssl rand -hex 32
3 CORS_ORIGIN ระบุ domain จริงเท่านั้น
4 mod_rewrite Enable ใน Apache config
5 php bin/genetics mutate Verify tables ก่อน go-live
6 .env ใน .gitignore ห้าม commit secrets เด็ดขาด

XAMPP Production: ตั้งค่า APP_URL ให้ตรงกับ domain จริง และเปิด mod_headers สำหรับ CORS headers
